Course Description:

The course introduces the student to programming techniques required to develop Web applications. Topics include HTML forms, JavaScript, Servlets and Java Server Pages, PHP, and MySQL, Web access to Oracle databases, and XML. Prerequisite CSc 1301 & 1302 (Principles of Computer Programming) CSC3410 (Data Structure)
